Soundcloud Linkify 0.1

Soundcloud Linkify identifies artists in tracklists for mixes and links to their SoundCloud pages

Install Extension View on GitHub

Hello

Soundcloud Linkify is a Chrome Extension that identifies artists mentioned in tracklists for mixes and links to their Soundcloud pages so you don't have to copy and paste artists' names into the search bar. You can get it in Chrome Store. It looks something like this

Known Issues

For some reason Chrome Store will not install the extension correctly unless you are logged in with your Google Account so you might want to do that before trying to install it.
